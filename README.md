## Instalasi app

1. buka folder app
2. rename file app_copy.ini menjadi app.ini
3. Isi data database anda pada app.ini

### Jika nama folder utamanya bukan FP

1. buka file .htaccess
2. ganti RewriteBase -nya jadi nama folder kalian
